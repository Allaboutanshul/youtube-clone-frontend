# YouTube Frontend Clone

A pixel-perfect, highly interactive YouTube Clone built purely with HTML5, CSS3, and modern Vanilla JavaScript. This project focuses on building a dynamic, client-side video feed interface that mimics the look, feel, and structural logic of the native YouTube desktop and mobile application experience.

## ✨ Key Features

- **Dynamic Video Rendering:** Leverages Vanilla JavaScript to dynamically inject video data matrices into customized gradient thumbnail grids complete with verification badges, video timestamps, and channel avatars.
- **Custom Video Player Modal:** A complete simulation of the YouTube player modal. It includes interactive custom controls such as standard Play/Pause functionality, Volume sliding/muting, 10-second fast-forward and rewind jumps, full-screen toggle, and an adjustable custom seek/progress bar.
- **Dark & Light Mode Integration:** Features a fully fluid, custom CSS variable theme-switcher accessible instantly via the header control interface.
- **Dynamic Content Filtering:** An fully interactive filter bar equipped with interactive chip navigation categorization tags (e.g., Tech, Gaming, Music, Travel) that handles list updates in real-time.
- **Skeleton Loader UI:** Integrated structural skeleton loader screen templates to ensure visual smoothness during content load states.
- **Fully Responsive Architecture:** Responsive layout transitions adapting from dual-axis collapsing persistent sidebars down to streamlined overlay menus optimized for high-performance mobile viewport dimensions.

## 🛠️ Built With

- **HTML5:** Semantic architecture layout structures (Headers, Asides, Main Content, Modals).
- **CSS3:** Custom properties (CSS variables), Flexbox, Grid configurations, and fine-tuned Media Queries for maximum cross-device scaling responsiveness.
- **JavaScript (ES6+):** Pure Vanilla implementation handles video simulation configurations, playback mechanics, modal lifecycle states, array filters, and user notification alert components.
- **Material Icons:** Clean, instantly recognizable iconography assets tracking modern design standards.
